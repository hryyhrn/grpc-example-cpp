# gRPC Client–Server Hello World (C++)

This repository contains a minimal client–server application using **gRPC** for communication.

## Build Instructions

1. **Set up gRPC** by following the official C++ quickstart guide:  
   https://grpc.io/docs/languages/cpp/quickstart/

2. **Add the required gRPC source and library paths** to the build script (`build.sh`).

3. **Build the project** by running: 
    ```bash 
    source build.sh 
    ```

## Steps to Run

1. **Navigate to the build directory** generated by build.sh.

2. **Start the server:** 
    ```bash 
    ./example_grpc_server 
    ```

3. **In a separate terminal, start the client** 
    ```bash
    ./example_grpc_client 
    ``` 
    This will create a client instance that invokes an RPC on the running server.
